# Tangent to the circle



Это первая лабораторная работа курса **Компьютерная графика**. В ней 
был реализован алгоритм построения касательных к окружности, 
исходящих из заданной точки. Отрисовка происходит в реальном времени 
без использования **OpenGL библиотек**(по заданию).
